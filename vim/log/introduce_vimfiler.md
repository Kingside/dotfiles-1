### UniteとVimfilerを連携

## Install 
```
// .vimrc
NeoBundle 'Shougo/vimfiler'

// in vim
:NeoBundleInstall
```


## Usage 
[ここを参照](http://qiita.com/hide/items/b0087bc0bbc36d0f91fb)

## VimFiler setting
```
// .vimrc
// vimfiler will be used as the default explorer (instead of netrw.)
let g:vimfiler_as_default_explorer = 1
```
